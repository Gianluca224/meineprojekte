
2 - HTML Listen - DEUTSCH
Erstelle eine Datei namens list.html
Diese Datei sollte die grundlegende HTML-Struktur enthalten
Füge eine ungeordnete Liste mit Artikeln hinzu, die du für dein Lieblingsrezept einkaufen müsstest
Füge eine geordnete Liste mit den Anweisungen für die Zubereitung des Rezepts hinzu
Füge ein Bild des Lieblingsessens unter das Rezept hinzu
Verlinke das Bild, um eine Website mit einem ähnlichen Rezept zu öffnen
Stelle sicher, dass das Bild angeklickt werden kann und sich die Website in einem neuen Tab zu öffnen
